# UI Wireframes

These are the low-fidelity wireframes for the Location-Aware Weather & Alerts Microapp.

## 1. Home Screen Wireframe
- Displays user location
- Shows weather icon, temperature, and condition
- Includes humidity, wind, and feels-like metrics
- “Alerts” card for navigating to alert details

## 2. Alert Popup Wireframe
- Shows a warning icon
- Displays a short alert message
- Includes a close (X) button
- Appears as an overlay or bottom toast notification
